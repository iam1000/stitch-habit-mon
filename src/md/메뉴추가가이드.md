# 📋 왼쪽 사이드바 메뉴 추가 가이드 (Add New Menu Guide)

HabitMons 앱의 왼쪽 사이드바에 새로운 메뉴(페이지)를 추가하기 위한 단계별 가이드입니다.
아래 표의 순서대로 작업을 진행하시면 에러 없이 메뉴를 추가하실 수 있습니다.

## 🛠️ 수정 및 생성 파일 목록

| 순서 | 구분 | 파일명 (경로) | 수정/생성 위치 | 설명 (작업 내용) |
| :--- | :--- | :--- | :--- | :--- |
| **1** | **수정** | `src/LanguageContext.jsx` | `translations` 객체 내부 | 메뉴 이름의 **다국어(한/영) 라벨**을 추가합니다.<br>*(예: `friends: "Friends"`, `friends: "친구들"`)* |
| **2** | **수정** | `src/MainLayout.jsx` | `menuItems` 배열 내부 | 사이드바 **메뉴 리스트**에 새 항목을 추가합니다.<br>*(예: `{ icon: Globe, label: t.friends, path: '/friends' }`)* |
| **3** | **생성** | `src/Friends.jsx` (예시) | `src/` 폴더 내 신규 생성 | 실제 화면에 보여질 **페이지 컴포넌트** 파일을 만듭니다.<br>*(Tip: 기존 `Dashboard.jsx` 등을 복사해서 이름만 바꾸면 빠릅니다)* |
| **4** | **수정** | `src/App.jsx` | **[중요!]** 상단 `import` 영역 | 새로 만든 컴포넌트를 **불러오는 코드**를 추가합니다.<br>`import Friends from './Friends';` |
| **5** | **수정** | `src/App.jsx` | `<Routes>` 내부 | 해당 메뉴 클릭 시 보여줄 **페이지 경로(Route)**를 연결합니다.<br>`<Route path="/friends" element={<Friends />} />` |

---

## 💡 작업 팁 (Tips)

1.  **아이콘 사용 (`MainLayout.jsx`)**
    *   `lucide-react` 라이브러리에서 제공하는 예쁜 아이콘들을 활용하세요.
    *   상단 `import { ... } from 'lucide-react';` 부분에 사용할 아이콘 이름(예: `Users`, `Heart`, `Star`)을 추가해야 합니다.

2.  **새 파일 생성 (`.jsx`)**
    *   기존 파일을 복사했다면, 반드시 내부의 **컴포넌트 이름**과 마지막 줄의 **`export default 이름;`** 부분을 새 파일명에 맞게 수정해주세요.

3.  **다국어 처리 (`LanguageContext.jsx`)**
    *   `en` (영어) 부분과 `ko` (한국어) 부분 모두에 키(Key)를 추가해야 언어 전환 시 에러가 나지 않습니다.
